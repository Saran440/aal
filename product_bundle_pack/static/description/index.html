Odoo 8: Product bundle

This module is used to Make bundle of the products.

How to use

  • Create bundle : from sale->product variants, here we have a one bundle tab where we can configure product for single bundle.


  • Sale quotation: user can add whole bundle in sale quotation, once it is confirmed sale order you can see delivery order option.


  • Delivery screen: You just have to select one bundle for sale order and related products for this bundle will be automatically added in delivery screen.
